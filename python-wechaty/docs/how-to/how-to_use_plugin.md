## 插件系统

插件系统提供了模块化的管理，能够让不同业务的代码隔离开，特别是针对于复杂的业务。

在处理不同业务时，通常选择将指定业务封装成一个插件，wechaty社区也欢迎大家贡献自己的插件，从而快速实现某些简单功能。
